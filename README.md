# 📱 QR Code Generator

A lightweight and user-friendly QR Code Generator built using **HTML, CSS, and JavaScript**.  
This application allows users to instantly generate QR codes for text, URLs, or any input directly in the browser.

---

## 🚀 Features

- Generate QR codes instantly
- Supports text, links, and custom input
- Simple and responsive UI
- Fast and lightweight
- No backend required
- Runs entirely in the browser

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 📂 Project Structure

```bash
QR-Code-Generator/
│── index.html
│── style.css
│── script.js
```

---

## ⚙️ How It Works

1. Enter text, URL, or any input
2. Click the **Generate QR Code** button
3. QR code is generated instantly
4. Scan using any QR scanner

---

## ▶️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Nandhu123-bangalore/QR-Code-Generator.git
```

2. Navigate to project folder:

```bash
cd QR-Code-Generator
```

3. Open `index.html` in browser

OR use Live Server in VS Code.
